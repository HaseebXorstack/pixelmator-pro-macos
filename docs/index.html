Installation Guide

Step-by-step installation via Terminal

  1. ⌘

    Open Terminal

    Press ⌘ + Space to open Spotlight, type Terminal and press Enter.

  2. 2

    Install Pixelmator Pro

    Paste this command and press Enter. Homebrew will download and install the app; you may be prompted for your Mac password.

    Terminal — Install Pixelmator Pro
    $brew install --cask pixelmator-pro
  3. ✓

    Done

    When the command finishes, open Pixelmator Pro from Applications or Spotlight (⌘ + Space).

About this app

Pixelmator Pro is a powerful, beautiful, and easy to use image editor designed exclusively for Mac with a stunning modern interface.

Pixelmator Pro is a professional-grade image editor with a beautifully designed interface — from ML-assisted edits to non-destructive workflows and RAW photos.

ML-Powered Tools

Use machine learning to automatically enhance photos, remove backgrounds, and upscale images.

Non-Destructive Editing

All edits are non-destructive, so you can change anything at any time.

Professional Color Adjustments

Powerful color adjustment tools including curves, levels, and color balance.

Vector Tools

Create scalable vector graphics alongside your pixel-based designs.
