## About this app

Pixelmator Pro is a professional-grade image editor with a beautifully designed interface.

- **ML-Powered Tools** — use machine learning to automatically enhance photos, remove backgrounds, and upscale images.
- **Non-Destructive Editing** — all edits are non-destructive, so you can change anything at any time.
- **Professional Color Adjustments** — powerful color adjustment tools including curves, levels, and color balance.
- **Vector Tools** — create scalable vector graphics alongside your pixel-based designs.
